# Project 1: Transformations

## How To Run

### Anaconda Setup
```
# Install Anaconda Environment
conda env create -f environment.yml
conda activate cvpj1
```

OR

### Pip Setup
```
# Install Modules Pip
pip3 install -r requirements.txt
```

### Run program
```
# Runs program, will write output images to output directory
python3 project.py
```
